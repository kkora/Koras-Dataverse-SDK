# Support

## Documentation

Start with the [documentation index](docs/index.md) — quick start, guides, configuration
reference, troubleshooting, and FAQ.

## Questions and help

- **GitHub Discussions** (preferred for how-do-I questions).
- **GitHub Issues** for reproducible bugs and feature requests — please use the issue templates.

When reporting a bug, include: package version(s), target framework, minimal reproduction, the
`DataverseError` details (category, HTTP status, error code, request id) with secrets redacted.

## What this is not

This is a community open-source project by Koras Technologies. It is not a Microsoft product;
for Dataverse platform issues, use Microsoft support channels (the `RequestId` from
`DataverseError` is what Microsoft support asks for).
